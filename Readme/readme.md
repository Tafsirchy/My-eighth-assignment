# App Name: Apps Vault

## Description:

Apps Vault is a simple app that allows users to install and manage their favorite apps. It provides a user-friendly interface for searching, installing, and managing apps. It also allows users to see the details of each app, such as its name, description, and rating.

## Technologies Used:

- React: A popular JavaScript library for building user interfaces.
- React Router: A library for handling routing in React applications.
- Axios: A library for making HTTP requests in JavaScript.
- Tailwind CSS: A utility-first CSS framework for building custom designs.
- DaisyUI: A library for building UI components in React.
- Lucid Icons: A library for using icons in React.
- React Toastify: A library for displaying toast notifications in React.
- Recharts: A library for building charts in React.
- Google Fonts: A library for using Google Fonts in React.
